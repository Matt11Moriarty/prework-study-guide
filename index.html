Git

  • Git Flow
    1. Create a branch
      • git checkout -b {branch name}
    2. Make changes
    3. Stage changes
      • git add -A
    4. Save the staged changes as commits
      • git commit -m "{commit message}"
    5. Pull the source code
      • git pull origin main
    6. Push all of the commits to your branch
      • git push origin {branch name}
    7. Create a PR in Github
    8. Address review comments
    9. Merge PR
    10. Delete your branch
